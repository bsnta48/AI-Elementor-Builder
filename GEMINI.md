# GEMINI.md

Context file for Gemini CLI.

## Read this first
Open **`PROJECT_STATUS.md`** at the repo root before doing anything — it is the live,
shared task state (done / in-progress / next) used by every AI CLI on this project.

## Before you stop
UPDATE `PROJECT_STATUS.md` with what you finished and what's next, so the next agent
(Claude Code, Codex, Cursor, Aider) can continue — especially if this session ends or
runs out of credits.

## Project docs
Architecture, build, and invariants are in `CLAUDE.md` / `AGENTS.md`. `PROJECT_STATUS.md`
is task state only, not documentation.
